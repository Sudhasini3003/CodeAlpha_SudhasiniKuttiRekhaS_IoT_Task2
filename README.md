 Light Monitoring System using Arduino

CodeAlpha IoT Internship – Task 2

Project Overview
This project demonstrates a Light Monitoring System using Arduino and an LDR (Light Dependent Resistor). The system automatically detects the surrounding light intensity and controls an LED based on the light level. The project was designed and tested using Tinkercad simulation.

Objectives
- To monitor ambient light intensity using an LDR sensor.
- To automatically control an LED based on light conditions.
- To understand sensor interfacing with Arduino.
- To simulate an IoT-based automation project using Tinkercad.

Features
- Automatic light detection
- LED turns ON in low light conditions
- LED turns OFF in bright light conditions
- Simple and low-cost automation system
- Tinkercad simulation

Components Used
- Arduino Uno
- LDR (Light Dependent Resistor)
- LED
- 220Ω Resistor
- 10kΩ Resistor
- Breadboard
- Jumper Wires

Software Used
- Tinkercad
- Arduino IDE

Working Principle
1. The LDR senses the surrounding light intensity.
2. Arduino continuously reads the analog value from the LDR.
3. If the light intensity is low, the LED turns ON.
4. If the light intensity is high, the LED turns OFF.
5. This process repeats continuously for automatic monitoring.
